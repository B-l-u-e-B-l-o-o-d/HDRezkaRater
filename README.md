## HDRezkaRater
Инструмент для массовой накрутки оценки на HDRezka

## Запуск
```bash
py HDRezkaRater.py -h
```
## Важно
- Корректная работа гарантирована только при использовании **ротационных удерживающих прокси**(Sticky IP)
- Прокси-файл должен содержать прокси построчно в формате `http://username:password@ip:port`
- Количество сделанных оценок будет равно количеству прокси
